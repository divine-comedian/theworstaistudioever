what it does, if anything

Three capabilities. All of them slower than the last guy.

We built a full communications stack and then removed the part that makes people anxious. The rest is here.

Unhurried Delivery

Outbound messages queue behind whatever the capybara is currently doing, which is nothing. Two tiers: "soon" and "when it gets there." We do not offer "now." Now is a human problem.

One Shared Colony Number

Every capybara on your account answers the same phone number. Private channels were considered and rejected. If one of them gets a text, all of them get the text. This is the product working correctly.

Co-Regulation Webhooks

No message fires until the herd collectively agrees it is a fine time. Each outbound event ships with a matching calm signal so nobody is ever startled. Your 2FA code arrives once everyone has settled.
